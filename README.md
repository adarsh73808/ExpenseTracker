💸 Expense Tracker
A full-stack Expense Tracker web application built with HTML, CSS, JavaScript, Node.js, and MongoDB. It allows users to manage and categorize their expenses and monitor their spending in real-time.

📌 Features
Add, edit, and delete expenses

Categorize expenses (Food, Travel, Bills, etc.)

Real-time total expense calculation

Responsive UI

Node.js & Express backend

MongoDB for persistent data storage

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB

🚀 How to Run Locally
1 Clone the repository
git clone https://github.com/adarsh73808/-Expense-Tracker.git
cd expense-tracker

2 Install dependencies :npm install

3Start MongoDB

Either run it locally (mongod)

Or use MongoDB Atlas (cloud)

4.Start the server
node app.js

5.Open your browser
Go to http://localhost:3000
